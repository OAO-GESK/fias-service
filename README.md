# Сервис получения сведений из адресного справочника ФИАС
- имя проекта: fias-service
- автор: max@huzm.ru
- дата: 2018

Сервис работает с БД "fias" formata Postgresql, где имена полей таблиц совпадают с именами параметров объектов выгрузки ФИАС XML-формата.
Имена таблиц:
- addrobj: адресные объекты
- house: дома
- stead: земельные участки
- room: помещения

## Требования
Версия python: 3.5+

## Зависимости
import:
- psycopg2
